## 祝福语生成器

### 使用步骤

1. **进入页面**
   - 点击首页的"春节祝福语"卡片
   - 或直接访问 `/blessings`

2. **输入需求**
   在输入框中描述你的需求，例如：
   ```
   生成一条给父母的传统祝福语
   ```
   ```
   给朋友的幽默新年祝福，要带点搞笑元素
   ```
   ```
   文艺风格的春节寄语，给喜欢文学的同学
   ```

3. **查看结果**
   AI 会生成包含以下信息的祝福语：
   - **标题**：祝福语的主题
   - **内容**：完整的祝福语文本
   - **风格标签**：标识祝福语的风格类型
   - **适用对象**：建议的使用对象

4. **使用祝福语**
   - 点击 **📋 复制** 按钮：将祝福语复制到剪贴板
   - 点击 **📤 分享** 按钮：通过系统分享功能发送

### 最佳实践

**明确指定对象**
```
✅ 好的示例：生成给领导的商务风格新年祝福
✅ 好的示例：给闺蜜的幽默祝福语
❌ 不好的示例：生成一条祝福语
```

**指定风格偏好**
```
✅ 好的示例：传统风格的拜年话，要包含成语
✅ 好的示例：文艺一点的新年寄语
❌ 不好的示例：随便写一条
```

**一次生成多条**
```
✅ 好的示例：生成3条给客户的商务祝福语
✅ 好的示例：我需要5条不同风格的新年祝福
```

## 表情包生成器

### 使用步骤

1. **进入页面**
   - 点击首页的"趣味表情包"卡片
   - 或直接访问 `/memes`

2. **描述创意**
   在输入框中描述你想要的表情包，例如：
   ```
   生成一个关于抢红包的搞笑表情包
   ```
   ```
   可爱风格的拜年表情包，要有小动物
   ```
   ```
   春节团圆吃饭的创意表情包
   ```

3. **查看方案**
   AI 会生成包含以下内容的完整方案：
   - **🎭 创意描述**：详细的画面和视觉描述
   - **💬 文案建议**：配套的搞笑或温馨文字
   - **📍 使用场景**：什么时候用这个表情包
   - **✨ 设计要点**：关键视觉元素和注意事项

4. **使用方案**
   - 点击 **📋 复制创意** 按钮：复制完整方案
   - 点击 **📝 复制文案** 按钮：只复制文案部分

## 常见问题

### Q: 生成的内容不满意怎么办？
A: 你可以：
1. 更详细地描述需求
2. 指定具体的风格和对象
3. 要求生成多个版本供选择
4. 基于已生成的内容提出修改意见

### Q: 可以同时生成多条吗？
A: 可以！在需求中说明数量，例如"生成3条祝福语"或"给我5个表情包创意"。

### Q: 生成的内容如何使用？
A: 
- **祝福语**：直接复制发送给亲朋好友
- **表情包**：将创意描述发给设计师，或使用 AI 绘图工具（如 Midjourney、DALL-E）生成图片

### Q: 支持哪些节日？
A: 当前版本专注于春节，未来会扩展到其他节日。

### Q: 生成的内容可以商用吗？
A: 生成的文字内容可以自由使用。如果要商用表情包，建议咨询法律顾问确认版权问题。

### Q: 没有生成我想要的风格？
A: 在输入时明确说明风格要求，或者提供参考示例，AI 会尽量满足你的需求。

### Q: 可以自定义生成的格式吗？
A: 可以在需求中说明格式要求，例如"要押韵"、"控制在50字以内"、"要有排比句"等。

## 💡 使用技巧

### 祝福语技巧
1. **明确对象关系**：说清楚对方是谁（家人、朋友、同事等）
2. **指定字数**：可以要求"50字以内的简短祝福"
3. **要求特殊元素**：如"包含2025蛇年元素"、"要有押韵"
4. **参考喜好**：如"对方喜欢古诗词，要文艺一些"

### 表情包技巧
1. **描述画面**：详细描述你想要的视觉效果
2. **指定角色**：如"用猫咪形象"、"卡通人物"
3. **说明情绪**：如"夸张搞笑"、"温馨可爱"
4. **参考风格**：如"简笔画风格"、"扁平插画风格"
