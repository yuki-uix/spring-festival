## ⚠️ 注意事项

### 图片生成
- 每次生成约需 **5-10 秒**
- 生成的图片链接 **24 小时后失效**，请及时下载
- 图片上会自动叠加清晰的文字水印
- 使用通义万相 API，请确保 `.env.local` 中已配置 `DASHSCOPE_API_KEY`

### CORS 问题
如果遇到 CORS 错误：
1. 刷新页面
2. 清除浏览器缓存
3. 重启开发服务器: `npm run dev`
