# 快速开始指南

## 🚀 3步开始使用

### 1️⃣ 启动应用
```bash
npm run dev
```
访问：http://localhost:3000

### 2️⃣ 选择功能
- **春节祝福语**：点击红色卡片 → `/blessings`
- **趣味表情包**：点击黄色卡片 → `/memes`

### 3️⃣ 输入需求
在聊天框中描述你的需求，AI 会立即生成内容！

---

## 💡 使用示例

### 生成祝福语

```
输入：生成一条给父母的传统祝福语
输出：包含传统成语和吉祥话的祝福语

输入：给朋友的幽默新年祝福
输出：轻松诙谐的新年祝福

输入：文艺风格的春节寄语
输出：优美抒情的诗意表达
```

### 生成表情包

```
输入：生成一个抢红包的搞笑表情包
输出：完整的创意方案（画面描述 + 文案 + 设计要点）

输入：可爱风格的拜年表情包
输出：萌系风格的表情包创意

输入：春节吃饺子的创意表情包
输出：美食主题的表情包方案
```

## 🎯 提示词技巧

### 好的提示词 ✅
- "生成3条给客户的商务新年祝福"
- "幽默风格的抢红包表情包创意"
- "给爷爷奶奶的传统拜年话，要包含健康长寿"

### 不够好的提示词 ❌
- "生成祝福语"（太笼统）
- "做个表情包"（缺少细节）
- "新年快乐"（没有明确需求）

## ❓ 遇到问题？

1. **没有生成内容**：检查 API Key 配置
2. **风格不对**：在提示词中明确说明风格
3. **内容不满意**：要求生成多个版本或提供更详细的描述
