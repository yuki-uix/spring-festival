## 📱 功能说明

### 祝福语卡片功能

每条生成的祝福语包含：
- **标题**：祝福语的主题
- **内容**：完整的祝福语文本
- **风格标签**：标识风格类型
- **适用对象**：建议的使用对象
- **📋 复制按钮**：一键复制到剪贴板
- **📤 分享按钮**：通过系统分享功能发送

### 表情包卡片功能

每个生成的表情包创意包含：
- **标题**：表情包主题
- **🎭 创意描述**：详细的画面和视觉描述
- **💬 文案建议**：配套的文字内容
- **📍 使用场景**：适合使用的场合
- **✨ 设计要点**：关键视觉元素列表
- **📋 复制创意**：复制完整方案
- **📝 复制文案**：只复制文案部分

## 🎯 使用技巧

### 提示词技巧

**✅ 好的提示词特征**
- 指定具体对象（父母、朋友、同事等）
- 说明风格偏好（传统、幽默、文艺等）
- 描述使用场景
- 提出特殊要求（字数、元素等）

**示例对比**

| ❌ 不够好 | ✅ 更好 |
|---------|--------|
| 生成祝福语 | 生成一条给父母的传统祝福语 |
| 做个表情包 | 生成抢红包手速快的搞笑表情包 |
| 新年快乐 | 给客户的商务新年祝福，要正式得体 |

### 互动技巧

1. **一次生成多个**
   ```
   生成5条不同风格的新年祝福
   ```

2. **提出修改意见**
   ```
   刚才那条祝福语能再幽默一点吗？
   ```

3. **指定具体要求**
   ```
   要包含"蛇年"元素，控制在50字以内
   ```

4. **参考喜好**
   ```
   对方喜欢古诗词，要文艺一些
   ```
