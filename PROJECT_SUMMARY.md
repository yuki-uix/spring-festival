## 🚀 如何使用

1. **启动应用**
   ```bash
   npm run dev
   ```

2. **访问页面**
   - 首页：http://localhost:3000
   - 祝福语生成器：http://localhost:3000/blessings
   - 表情包生成器：http://localhost:3000/memes

3. **生成内容**
   - 在聊天框中输入需求
   - AI 会根据需求生成相应内容
   - 点击复制或分享按钮使用

## 🔄 可扩展性

代码结构支持轻松扩展：

1. **添加新风格**：在 `styleConfig` 中添加新的风格配置
2. **添加新组件**：创建组件并在 `tambo.ts` 中注册
3. **修改 AI 行为**：调整页面中的 `systemPrompt`
4. **扩展到其他节日**：复制页面结构，修改主题和提示词
