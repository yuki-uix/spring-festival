## 🌟 功能特性

### 1. **智能建议按钮**
用户与 AI 对话后，会自动显示 3 个智能建议按钮：

```
┌──────────────────────────────────────┐
│ AI: 我已经为您生成了祝福语...       │
└──────────────────────────────────────┘
    ↓
[再生成一条] [换个风格] [生成图片]  ← 智能建议
```

### 2. **初始建议**
当用户第一次进入对话时，显示默认建议：

```
欢迎！👋

[快速开始] [查看示例] [了解功能]  ← 初始建议
```

### 3. **键盘快捷键**
用户可以使用键盘快速选择建议：

- **Mac**: `⌘ + ⌥ + 1/2/3`
- **Windows/Linux**: `Ctrl + Alt + 1/2/3`

### 4. **自动生成**
AI 会根据对话上下文自动生成相关的建议：

**对话示例**：
```
用户: "生成传统风格的祝福语"
AI: [显示传统祝福语]

智能建议:
[生成幽默风格] [再来一条] [生成表情包]
```

## 🎨 工作原理

### 技术实现

```typescript
// MessageThreadFull.tsx
<MessageSuggestions initialSuggestions={defaultSuggestions}>
  <MessageSuggestionsList />  ← 显示建议按钮
</MessageSuggestions>

// MessageSuggestions.tsx
const {
  suggestions,          // AI 生成的建议
  accept,              // 接受建议的函数
  isGenerating,        // 是否正在生成
} = useTamboSuggestions({ maxSuggestions: 3 });
```

### 自动化流程

1. **用户发送消息** → AI 处理
2. **AI 回复完成** → 自动触发建议生成
3. **生成 3 个建议** → 显示为可点击按钮
4. **用户点击建议** → 自动发送消息
5. **循环继续** ↻

## 🐛 常见问题

### 1. 建议不显示？

**检查**：
- 是否已发送至少一条消息？
- 网络连接是否正常？
- 浏览器控制台是否有错误？

### 2. 建议不相关？

**原因**：AI 根据对话上下文生成，可能需要更多对话轮次。

**解决**：多对话几轮，AI 会更了解你的需求。

### 3. 键盘快捷键不起作用？

**检查**：
- 是否按对了组合键？（Mac: ⌘+⌥+数字，Windows: Ctrl+Alt+数字）
- 输入框是否失焦？
